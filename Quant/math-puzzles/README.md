## Common Problems

### Easy
1. Sum of first n natural numbers
2. Number of handshakes for n people
3. Clock angle at given time
4. Area of shapes (circle in square, etc.)

### Medium
1. **Infinite Series**: Sum of 1 + 1/2 + 1/4 + 1/8 + ...
2. **Digit Problems**: How many times does digit 1 appear from 1 to 1000?
3. **Weighing Problems**: Find fake coin in n coins with k weighings
4. **Calendar Math**: What day of week will it be in 100 days?
5. **Rectangle Tiling**: Ways to tile 2×n board with 2×1 dominoes

### Hard
1. **Monty Hall Variant**: n doors, k prizes, optimal strategy
2. **Geometric Optimization**: Maximize area with fixed perimeter
3. **Expected Value**: Expected number of coin flips to see HH vs HT
4. **Number Theory**: Prove √2 is irrational
5. **Combinatorial**: Number of ways to climb n stairs taking 1 or 2 steps

## Problem-Solving Strategies

### Pattern Recognition
- Look for sequences
- Try small cases first
- Build up to general solution

### Algebraic Approach
- Define variables clearly
- Set up equations
- Solve systematically

### Visual Thinking
- Draw diagrams
- Use geometric intuition
- Visualize the problem

### Special Techniques
- Proof by contradiction
- Mathematical induction
- Symmetry arguments
- Invariants

## Tips for Interviews
- Start with special/simple cases
- Look for patterns
- Use concrete examples
- Check dimensional analysis
- Verify with extreme values
- Express answer in simplest form
