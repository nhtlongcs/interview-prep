- Sampling and estimation
- Confidence intervals
- Hypothesis testing
- p-values and significance levels
- Type I and Type II errors
- Regression analysis
- Maximum likelihood estimation
- Central limit theorem applications

## Example Problems

### Easy
1. Calculate mean, median, and standard deviation of a dataset
2. What is the 95% confidence interval for a sample mean?
3. Interpret correlation coefficient

### Medium
1. A/B test: how to determine if difference is significant?
2. Explain Type I vs Type II errors with examples
3. When to use t-test vs z-test?
4. How does sample size affect confidence interval?

### Hard
1. Design an experiment to test effectiveness of a new trading strategy
2. Explain multiple testing problem and correction methods
3. How would you detect if two distributions are different?
4. Regression to the mean in portfolio performance
